4.1刷题技巧：五遍刷题法
第一遍：5分钟读题+思考，没有思路的情况，直接看解法，背诵、默写好解法（很重要）
第二遍：马上自己写-->LeeCode提交，多种解法比较、体会，优化
第三遍：过了一天后，再重复做题。根据不同解法的熟练程度进行专项练习
第四遍：过了一周，反复回来练习相同题目
